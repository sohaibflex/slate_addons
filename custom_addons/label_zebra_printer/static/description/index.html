Label Printing From Zebra Printer module, which allows to send labels, barcodes etc. directly to the Zebra printers.

Note: The purpose of this module is only to send the labels directly to the zebra printer. The formatting is not guaranteed or
it has to be customized.
Please First Install QZ Tray From Here: https://qz.io/download

**This module uses a 3rd party java-script client called qz which requires a certificate to whitelist your domain so that you
don't receive any pop-ups and can work hassle free. You can either buy the certificate from qz or use a self signed
certificate. For more information, kindly visit.https://qz.io/wiki/generate-certificate

Key Features

Send direct commands to Zebra Printer.
